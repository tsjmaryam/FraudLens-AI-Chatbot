# Fraud AI Chatbot 
The Fraud AI Assistant Chatbot helps fraud analysts make faster, fairer, and more transparent decisions by combining interpretable machine learning models with natural-language summaries. Built using Python, EBM, it offers explainable outputs, bias detection, and a user-friendly chat interface to support real-world fraud investigations
