## Requirements Explanation

The following libraries were used in FraudLens:

- **streamlit** – interactive web interface for the chatbot and fraud scoring.
- **pandas** – data cleaning and transformation for the transaction and user datasets.
- **numpy** – numerical operations supporting preprocessing and modeling.
- **scikit-learn** – train/test splitting, evaluation metrics, and preprocessing tools.
- **interpret** – training and explaining the Explainable Boosting Machine (EBM) model.
- **joblib** – saving and loading trained models efficiently.
- **python-dotenv** – securely loading environment variables (e.g., API keys).
- **openai** – generating natural-language explanations and RAG-based responses.
- **matplotlib** – model evaluation plots (ROC, distributions).
- **seaborn** – enhanced statistical visualizations for EDA.
- **tqdm** – helpful progress bars during heavy computations.
